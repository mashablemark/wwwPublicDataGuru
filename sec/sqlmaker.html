SQL Data Definition Language (DDL) derived from XBRL Filings without Notes distributed with each quarterly Financial Statement Data Set available for download from https://www.sec.gov/dera/data/financial-statement-data-sets.html.

Also see DDL for XBRL Filings with Notes.

After the DDL is created and run, the files can be downloaded and unzipped from the Edgar filings data ingest page.
